# Dano

Dado de dano x MBA = dano na vida do robô

M.B.A. = Multiplicador Base de Ataque = valor que multiplicado aos dados no jogo vão resultar no dano aos alvos

MBA é diferente pra cada arma

Multiplicador adicional = atributo mod.pwr + talento de adição de dano + estado de energia

Armas de energia -> MBA = 8 a 12
Armas físicas -> MBA = 5 a 8

para cada estado de energia apartir do nível 2 -> soma +2 no multiplicador adicional excete no último nível que é somado apenas +1
Exemplo:
Nível 1 -> MBA = -2
Nível 2 -> MBA = 0
Nível 3 -> MBA = +2
Nível 4 -> MBA = +4
Nível 5 -> MBA = +5
